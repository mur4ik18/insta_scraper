# Insta_scraper

This script get inforamtion about open instagramm account.

## Installation
Clone git
```bash
git clone https://github.com/mur4ik18/insta_scraper
```


Use the package manager [pip](https://pip.pypa.io/en/stable/) to install:
* requests
* json

```bash
pip install requests json
```

## Usage

```bash
python index.py
```
